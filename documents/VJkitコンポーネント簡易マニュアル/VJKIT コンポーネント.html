VJKitのコンポーネント簡易マニュアル

VJKitについて

https://github.com/unity3d-jp/fuze-vj-kit


コンポーネントのアタッチ方法

GameObjectを選択（Or作成）　＞　Add Componentをクリック　＞　アタッチしたいオブジェクトを選択する




System

シーンに一つ（もしくは複数）配置する、ModifiresやTriggerを動かす情報源。


※設定が面倒な場合はAssets>VJKit>Preset Items>VJのプレハブを利用する。

対応コンポーネント

VJManager（Sourceに指定したAudioSourceを解析し音の周波数帯を管理する）
     ・FFT（）
     ・BandLevels（帯域の値。実行時VJDataSourceから自動で取得する）
     ・Debug Window（周波数帯の動作状況を確認するデバッグウィンドウを表示する）
     ・Use Microphone（マイクから音を取得する）

VJDataSource(Spectrum)（周波数帯の情報を設定する）
     ・ Source Name（帯域名。ModifiersやTrigerではこの名称を指定する）
     ・Boost（帯域の値を増幅する）
     ・lowerBand（周波数の最低値）
     ・upperBand（周波数の最大値）

VJDataSource(Volume)（音量の情報を設定する）


Modifiers 

オブジェクトにアタッチし、サウンドに合わせて動かす（常に動作する）



基礎パラメータ

VJManager（音の周波数帯を管理するマネージャー）
source（音の周波数帯）
boost（値の増幅値）
variationAmount（変化量のゆらぎ）
variationSpeed（変化速度のゆらぎ）
variationOcterb（オクターブのゆらぎ）
limitMinMax（min/maxに値を絞るかどうか）
valueMin（値の最小値）
valueMax（値の最大値）
middleOffset（中央値のオフセット）
rest（値を中央値に徐々に戻すかどうか）
restStrength（戻す際の強さ）
negative（値を反転するかどうか）
multiple（複数のオブジェクトに対する操作かどうか）


対応コンポーネント

Animator（アニメーション速度を変更する）

GameObject Property（Property to Modifyに従いアタッチしたオブジェクトを動かす）

Light Property（Property to modifyに従いアタッチしたオブジェクトのライトを制御する）

Material Property （Property to Modify及びProperty Nameに従いマテリアルの色を変更する）
Scene Speed（シーンの全体の速度を変化させる。ModifiersやTrigger等のサウンドが影響しないコンテンツに有効）



Trigger

オブジェクトにアタッチし、サウンドに反応して動かす（トリガー発動時のみ動作）



基礎パラメータ

VJManager（音の周波数帯を管理するマネージャー）
source（音の周波数帯）
boost（値の増幅値）
variationAmount（変化量のゆらぎ）
variationSpeed（変化速度のゆらぎ）
variationOcterb（オクターブのゆらぎ）
limitMinMax（min/maxに値を絞るかどうか）
valueMin（値の最小値）
valueMax（値の最大値）
middleOffset（中央値のオフセット）
rest（値を中央値に徐々に戻すかどうか）
restStrength（戻す際の強さ）
negative（値を反転するかどうか）
multiple（複数のオブジェクトに対する操作かどうか）
ThresholdType（トリガー発動のタイプ）
          ・Value_LessThan（値がthresholdより小さい）
          ・Value_MoreThan（値がthresholdより大きい）
          ・Difference_LessThan（（現在の値-前回の値）の値がthresholdより小さい）
          ・Difference_MoreThan（（現在の値-前回の値）の値がthresholdより大きい）
threshold（トリガー発動の基準値）


対応コンポーネント

Animator Transition（トリガー発動時、Animationを切り替える）
Animatior（トリガー発動時、アニメーションを動かす。アニメーション速度はThresholdに依存する）
Game Object Property（トリガー発動時、Property to Modifyに従いアタッチしたオブジェクトを動かす）
Light Property（トリガー発動時、Property to modifyに従いライトを制御する）

Material Property（トリガー発動時、Property to Modify及びProperty Nameに従いマテリアルの色を変更する）

Send Message（トリガー発動時、SendMessageを対象のオブジェクトに送信する。オブジェクトが設定されていない場合、アタッチしたGameObjectに送信する）

Emit Particle（Ol Particle system）（トリガー発動時、パーティクルを生成する。古いパーティクルシステムに対応）

Emit Particle（Shuriken）（トリガー発動時、パーティクルを生成する。Shurikenに対応）

On Off Trigger（Component to On Offで指定したコンポーネントのOn/Offを切り替える）
Spown Object  （トリガー発動時、randomPositionOffsetに指定した座標にオブジェクトを生成する。destroySpawnedObjectが指定されている場合、objectLiveSec秒後に自壊する）



Utilities

その他便利機能

対応コンポーネント

Alway Look at camera （アタッチしたオブジェクトは常にカメラの方を向く。camを指定しなかった場合、main Cameraを指定する）
Fbm Motion（Fbm Motionのチェックが入っている際、カメラの手振れを再現する）
Destroy object in sec（アタッチしたオブジェクトを[ObjectLiveSec]秒後に消滅させる）
Freeze Position（座標・回転・大きさを固定する）
Web Camera（Web Cameraで撮影した内容を指定のマテリアルのテクスチャに設定する。モバイルの場合はカメラで撮影した内容になる）

Wirefame Camera（画面をワイヤーフレームモードにする）

MoviePlayer（ムービーを再生する。音は出ない）

Cameras：カメラにアタッチし、主にカメラを制御する

Follow Camera（対象を追跡する）
Rotate Camera（対象を中心に回転する）
